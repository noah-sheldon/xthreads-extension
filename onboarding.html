Get Your API Key

To use xThreads Agent, you'll need an API key from your xThreads.app account.

How to get your API key:

  1. Visit xthreads.app/profile
  2. Sign in to your account (or create one if needed)
  3. Navigate to the "API Keys" section
  4. Copy the key and paste it below
2

Configure Keywords

Set up keywords that the agent will look for when finding tweets to reply to.

Tips:

  • Use relevant keywords for your niche or interests
  • Include both broad and specific terms
  • The agent will find tweets containing any of these keywords
  • You can always change these later in the extension popup
3

Select Reply Tone

Choose the tone for your AI-generated replies. You can change this anytime.
